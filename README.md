# GIT Functions

[PT] Lib para funções do git.<br>
[EN] Lib for git functions.


## Funcionalidades:
* Checar se existem modificações

## Como usar:

### Instalação
```
    pip install gitfunctions
```

### Exemplo
```
    from gitfunctions import gitfunctions

    git = gitfunctions.Git(repo_path='C:\\pasta_projeto')
    git.check_updates()
```
